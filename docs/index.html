JupyDash

Drag a Jupyter notebook in. Get a live, modular dashboard of its outputs.

Download for macOS

Universal (Apple Silicon + Intel) · macOS 11+ · Signed & notarized.

  • 1. Open the downloaded .dmg and drag JupyDash to Applications.
  • 2. Launch it, then drag a .ipynb onto the window.
  • 3. Its plots, tables and values appear as draggable panels.
  • Notebooks run in your own Jupyter kernel — you'll need Python with ipykernel installed (any Anaconda / Jupyter user already has it).
