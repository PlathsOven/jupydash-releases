# JupyDash — downloads

Public download host for **[JupyDash](https://github.com/PlathsOven/jupydash)**.

### Download
- **macOS (universal):**
  https://github.com/PlathsOven/jupydash-releases/releases/latest/download/JupyDash-macOS.dmg
- **Windows (x64):**
  https://github.com/PlathsOven/jupydash-releases/releases/latest/download/JupyDash-Windows-Setup.exe

The Windows installer is built by the [`build-windows` workflow](.github/workflows/build-windows.yml) on a native Windows runner (the PyInstaller sidecar can't be cross-compiled); the macOS DMG is built and notarized on a Mac.

Landing page: https://plathsoven.github.io/jupydash-releases/

Notebooks run in your own Jupyter kernel (Python + `ipykernel`).
